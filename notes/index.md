# Zhanshi Research Board

这里不是通用论文首页，而是我的研究看板。

## 当前关注

- 医学影像
- 眼科 AI
- Agent 系统
- 计算机视觉
- 持续学习

## 我希望每天回答的问题

1. 今天出现了哪些和我研究主线直接相关的论文？
2. 哪几篇值得进入精读，而不是只看摘要？
3. 哪些方法、数据设定或实验结论值得纳入后续项目？
4. 哪些工作只是在热点里重复堆料，应该快速跳过？

## 看板使用方式

- 左侧按日期查看 daily
- 先看当天 `index.md` 的精选结果
- 再进入具体论文笔记做全文阅读
- 对能继续追的方向，转入 `ideas/`

## 当前默认筛选偏好

- 高优先级：眼科 AI、医学影像、持续学习、CV 核心任务、Agent
- 重点信号：外部验证、泛化能力、临床场景、incremental learning、tool use
- 低优先级：survey、纯 benchmark、纯 leaderboard、纯数据集整理
